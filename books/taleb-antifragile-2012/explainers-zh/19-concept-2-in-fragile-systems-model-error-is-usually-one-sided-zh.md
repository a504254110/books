## 核心意思

在 fragile system 里，model error 往往不是正负抵消，而是偏向单边伤害。

## 详细解释

Taleb 要区分两类错误。有些错误像交易里下单数量弄错，长期看可能有时帮你、有时害你，彼此抵消。

但另一些系统不是这样。交通、战争、复杂政策、延误系统里，误差通常不会平均抵消，而是更容易积累成坏结果。

所以在 fragile system 里，`模型只错一点点` 不是安慰，因为 error 的成本本身就是 asymmetric 的。

## 这个例子到底在说明什么

交易里把一千单位误下成两千，可能赚也可能赔；但在交通或机场系统里，小失误通常不会带来对称的好处，而是更可能让拥堵和延误更糟。这个例子说明的是：有些系统里的 error 是 neutral 的，有些则是 one-sided 的，后者更危险。
