# Source materials (private / local)

This folder is for **your own** licensed inputs. The public repository **does not** include:

- The commercial PDF of *Antifragile*
- Machine exports of full book text (e.g. `extracted-pages.json`)

## What you may add locally (never commit)

Place files here only on your machine, for verification and future editing:

- Your legally obtained PDF (any filename you prefer)
- Optional extracts or notes you generate for personal use

Git ignores `*.pdf` and `extracted-pages.json` under `books/*/source/` so they are not pushed by mistake.

## What stays in the repo

- `chapter-01-notes.md`, `front-matter-notes.md`, and similar files are **reader-produced grounding notes** (page maps and working summaries), not a substitute for the book.

If you clone this vault for reading only, you do not need anything here beyond what is already tracked.
